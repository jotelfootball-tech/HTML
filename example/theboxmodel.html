The Box Model

In CSS, every element is represented as a rectangular box. The CSS box model describes the rectangular boxes that are generated for elements in the document tree and consists of: margins, borders, padding, and the actual content.

Components of the Box Model

  • Content: The innermost part where text and images appear.
  • Padding: Space between the content and the border. It clears an area around the content.
  • Border: A line surrounding the padding (if any) and content.
  • Margin: The outermost space that separates the element from other elements.
